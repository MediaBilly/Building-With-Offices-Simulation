Για τη μεταγλώττιση του προγράμματος γράψτε javac AskisiJava.java

Σχόλια:
1)Για να αξιοποιήσω και εδώ την κληρονομικότητα έφτιαξα μία κλάση Person και για τον επισκέπτη(Visitor) και τον συνοδό(Guest) έφτιαξα τις αντίστοιχες ξεχωριστές κλάσεις οι οποίες
κληρονομούν τον Person.
2)Για την χωρητικότητα των γραφείων πήρα έναν επιπλέον περιορισμό έτσι ώστε να είναι πάντα >=2.Αυτό το έκανα για να μπορούν οι επισκέπτες που έχουν συνοδό να μπαίνουν ταυτόχρονα με τον
συνοδό τους στο αντίστοιχο γραφείο.
